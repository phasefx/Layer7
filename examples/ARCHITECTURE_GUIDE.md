# Codebase Architecture Guide

## Executive Summary

The Layer7 codebase implements a sophisticated layered architecture for parsing, structuring, and executing multi-language code compositions. Data flows through a well-defined pipeline where `Layer7Parser` in `core_structure.py` processes text input into structured `HeaderNode` objects, which are then managed by the `AddressResolver` in `addressing.py` for cross-referencing and navigation. The `CompositionEngine` in `flow_control.py` orchestrates execution flows based on routing rules, while the `MCPDispatcher` in `language_integration.py` handles actual code execution across multiple programming languages through a standardized interface.

The core state management relies on a combination of object-oriented composition and centralized state tracking. The `AddressResolver` maintains an internal registry of nodes through its `register_node` method, creating a graph-like structure that enables efficient resolution of references. The `CompositionEngine` manages execution state through `execute_chain` and `execute_composition` methods, maintaining cursor positions and execution context in variables like `current_val` and `state`. The `MCPDispatcher` maintains temporary file state and execution results through `ExecutionResult` objects, creating a consistent interface for language-specific execution.

Hidden architectural coupling exists primarily through the tight integration between `Layer7Parser` and `AddressResolver` through shared data structures, where `HeaderNode` objects contain references that `AddressResolver` expects to resolve. Additionally, `CompositionEngine` directly manipulates `HeaderNode` state through `execute_chain`, creating implicit dependencies between flow control and structural parsing. The `MCPDispatcher` also couples to `Layer7Parser` through its reliance on `HeaderNode` structure for code generation, and the `title_to_identifiers` function creates a dependency between naming conventions and identifier resolution that isn't immediately apparent from the public interfaces.
